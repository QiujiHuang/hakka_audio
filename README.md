# Hakka audio stimuli

This repository contains the Yuancheng--Zhongxin Hakka audio stimuli and the corresponding experimental materials used in the forced-choice experiment reported in the study *Questions in Yuancheng-Zhongxin Hakka*.

Although the repository is named `hakka_audio`, it also includes the spreadsheet and CSV files used for the Yuancheng-Zhongxin Hakka experiment.

## Contents

This repository includes:

- `Hakka_materials.xlsx`: spreadsheet containing the Yuancheng-Zhongxin Hakka target items, filler items, and practice items.
- `itemscibles.csv`: CSV file containing the Yuancheng-Zhongxin Hakka target items.
- `fillers.csv`: CSV file containing the Yuancheng-Zhongxin Hakka filler items.
- `practice.csv`: CSV file containing the Yuancheng-Zhongxin Hakka practice items.
- `.wav` files: audio stimuli used in the Yuancheng-Zhongxin Hakka experiment.

## File naming convention

The audio files are named according to the following system:

- `HK` = Yuancheng-Zhongxin Hakka
- item number = experimental item number
- `P` = practice item
- `F` = filler item
- `Q` = question stimulus
- `A` = answer stimulus
- `Focus` = focus condition
- `Topic` = topic condition
- `TopicM` = marked topic condition
- `Preference` = preference question in filler items
- `Left` / `Right` = answer position in filler items

For example:

- `HK-1-Q-Focus.wav` refers to Hakka item 1, question stimulus, focus condition.
- `HK-1-Q-Topic.wav` refers to Hakka item 1, question stimulus, topic condition.
- `HK-1-Q-TopicM.wav` refers to Hakka item 1, question stimulus, marked topic condition.
- `HK-1-A-Focus.wav` refers to Hakka item 1, answer stimulus, focus condition.
- `HK-1-A-Topic.wav` refers to Hakka item 1, answer stimulus, topic condition.
- `HK-F8-Q-Preference.wav` refers to Hakka filler item 8, preference question.
- `HK-F8-A-Left.wav` and `HK-F8-A-Right.wav` refer to the left answer for filler item 8.
- `HK-P1-Q-Focus.wav` refers to Hakka practice item 1, question stimulus, focus condition.

## Audio format

All audio stimuli are provided in `.wav` format.

## Experimental materials

The file `Hakka_materials.xlsx` provides an overview of the Yuancheng-Zhongxin Hakka experimental materials. The CSV files are provided for accessibility and can be used directly with the PCIbex experimental scripts.

The materials include:

- target items;
- filler items;
- practice items;
- corresponding question and answer stimuli;
- condition labels used in the experiment.

## Related thesis

These materials were prepared for the Yuancheng-Zhongxin Hakka forced-choice experiment in the study:

*Questions in Yuancheng-Zhongxin Hakka*

## Notes

The repository is intended to make the experimental materials and audio stimuli accessible for verification and reference. No participant data are included in this repository.
